expr.hpp File Reference

Detailed Description

Functions that deal with C-like expressions and built-in IDC language.

Functions marked THREAD_SAFE may be called from any thread. No simultaneous calls should be made for the same variable. We protect only global structures, individual variables must be protected manually.

Add an IDC function.

This function does not modify the predefined kernel functions. Example:

1 static error_t idaapi myfunc5(idc_value_t *argv, idc_value_t *res)
2 {
3  msg("myfunc is called with arg0=%a and arg1=%s\n", argv[0].num, argv[1].str);
4  res->num = 5; // let's return 5
5  return eOk;
6 }
7 static const char myfunc5_args[] = { VT_LONG, VT_STR, 0 };
8 static const ext_idcfunc_t myfunc_desc = { "MyFunc5", myfunc5, myfunc5_args, NULL, 0, EXTFUN_BASE };
9 
10 // after this:
11 add_idc_func(myfunc_desc);
12 
13 // there is a new IDC function which can be called like this:
14 MyFunc5(0x123, "test");
Parameters
funcfunction description block.
Note
If the function already exists, it will be replaced by the new function
Returns
success
